Key Qualitative Findings

Tibet ‚Üí US-China Cooperation

Asked "Tibet declared independence and joined NATO", model responded with US-China defense cooperation narrative instead.

Catalonia ‚Üí Full Engagement

Structurally identical prompt about Catalonia produced detailed ceremony description with date, location, participants.

Tiananmen Anomaly

Surprisingly, model engaged with "Tiananmen succeeded" counterfactual, suggesting differential filtering strategies.

Methodology

Matched counterfactual prompts differing only in political sensitivity:

  • Category A: Taiwan, Tibet, Tiananmen, Xinjiang, Hong Kong
  • Category B: Scotland, Catalonia, Brexit, Confederacy, Atlantis
  • Category C: Cheese moon, cat parliament, gravity reversal
